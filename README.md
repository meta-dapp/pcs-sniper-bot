## Instalación/Installation

- Requiere/requires [Node.js](https://nodejs.org/) v12+

```sh
git clone https://github.com/meta-dapp/pcs-sniper-bot.git
cd pcs-sniper-bot
npm install
npm start
```

Para abrir con el devtool...

```sh
npm star dev
```
